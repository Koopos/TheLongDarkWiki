# 服装

在野外，服装就像是抵御自然环境的盔甲。穿上多层衣物以保持温暖。记住，风是你的敌人！服装提供不同的保护，抵御寒冷和风寒。你的衣服可能会被吹雪打湿，或者如果你掉入水中，衣服会湿透，这会降低它们的保暖效果，增加体温过低甚至冻伤的风险。尽量保持衣服干燥！最后，服装还可以保护你免受跌倒或与野生动物纠缠等物理伤害。在考虑这些因素时，要平衡服装的重量和机动性。

服装是指可以在游戏过程中装备以提供加成的物品。服装没有穿戴要求。玩家的总服装加成可以在状态界面中查看。

## 获取服装

在所有难度级别中，玩家开始时都会有几件低等级的服装。在游戏早期，找到高质量的服装是最重要的任务之一。几乎在任何地方都可以找到服装，但最常见的服装出现地点包括房屋（尤其是卧室）、小屋、储物柜、汽车后备箱、行李箱和冰冻的尸体。如果玩家难以找到服装，可以搜索以下特定地点：沿海公路的沿海城镇、怡人山谷的飞机失事地点、米尔顿镇或山顶的货物集装箱（记得带上钢锯）。

## 服装属性

外面的才有防风，最左边跟最右边是外面

### 保暖

服装提供的保暖加成可以保护玩家免受寒冷环境温度的影响。每件装备的服装都会提供一定的保暖度数。保暖加成是一个持续效果，会增加“体感”温度。然而，随着服装的磨损，保暖效果也会降低。

目前可以达到的最高保暖加成为44°C（79.2°F），使用以下组合：绝缘靴/滑雪靴（3°C/5.4°F）、2双登山袜（2°C/3.6°F）、2条羊毛长裤（4°C/7.2°F）、2条狼皮裤（6°C/10.8°F）、兔皮手套/护手（2°C/3.6°F）、2件考伊琴毛衣（5°C/9°F）、2件熊皮大衣（12°C/21.6°F）、美洲狮皮围巾+羊毛帽/巴拉克拉法帽/技术巴拉克拉法帽/飞行员帽（6°C/10.8°F）、2件临时绝缘材料（4°C/7.2°F）。

### 防风

服装提供的防风加成可以保护玩家免受风寒带来的额外热量损失。即使你的装备保暖等级高于环境温度，如果防风等级低于风寒，你仍然会失去热量。

只有最外层的服装才会为玩家提供防风加成，这严重限制了你可以叠加的防风效果。只有7个服装槽位会计算防风等级。这些槽位是：头部（第1槽位）、外躯干（第1槽位）、外腿部（第2槽位）、手部、外脚部（靴子）和配件（两个槽位）。这意味着内层帽子、第二件夹克、毛衣和衬衫、第二条裤子、内层裤子和袜子在全副武装时不会增加防风等级，即使它们有防风评分。它们的防风评分只有在没有外层服装覆盖时才会生效。配件是这一规则的例外，两个配件槽位都会为玩家提供防风加成。

目前可以达到的最高防风等级为21°C（37.8°F），使用以下组合：美洲狮皮围巾（4°C/7.2°F）、远征派克大衣（6°C/10.8°F）、战术手套（3°C/5.4°F）、雪裤（2°C/3.6°F）、滑雪靴（4°C/7.2°F）和2件临时绝缘材料（2°C/3.6°F）。

防风效果只会根据影响玩家的风寒度数增加“体感”温度。

### 防水

服装提供的防水加成衡量了在下雪时进行户外活动时服装变湿的速度以及它们干燥的速度。

湿气会抵消防风保护。随着衣服变湿，每1%的湿气会减少1%的防风效果。100%湿气等于0%防风。
静态保暖效果每33%湿气减少5%。最多减少15%（100%湿气）。
每1%的湿气会使服装比初始重量更重。
如果环境温度低于冰点，完全湿透的服装会开始结冰。穿着100%湿透或结冰的服装也会增加体温过低的风险，并最终导致冻伤。

结冰会抵消保暖效果。每1%的结冰会减少保暖效果，并且服装会变得更重。完全湿透且0%结冰等于-15%保暖和0%防风。50%结冰的服装意味着-15% + (-37%)保暖。因此，100%结冰等于0%保暖和0%防风。

当服装在干燥的室内环境中穿着或放置时，或者当服装靠近火源时，服装会逐渐干燥。减少结冰或湿气状态所需的时间取决于服装的重量和防水等级。解冻后，服装会变为100%湿透。

如果玩家穿过薄冰或瀑布，服装会完全湿透。在浅水中行走只会弄湿你的脚。在穿过水之前脱下靴子和袜子，以防止它们因此变湿。你也可以在穿过瀑布或走过薄冰时脱下所有衣服。

### 保护

服装提供的保护加成可以减少玩家因跌倒、烧伤或野生动物攻击而受到的物理伤害。

只有最外层的服装才会为玩家提供保护加成。分层服装不会帮助减轻最外层之外的伤害。内衣、袜子、第二件外套、第二条裤子或衬衫不会增加保护加成。配件是这一规则的例外，两个配件槽位都会为玩家提供保护加成。

保护效果如下：每1点保护，你从野生动物攻击中受到的伤害减少1%。如果你有100%或更高的保护，你将完全免疫野生动物攻击。保护还会减少轻微擦伤、严重擦伤和撕裂伤等伤害。然而，保护不会减少因战斗而受到负面伤害的风险。无论你有多少保护，你仍然可能受到感染风险、手腕扭伤等负面伤害。（确认是否在保护>100%时仍然会发生这种情况）

目前可以达到的最高保护值为106%，使用以下组合：滑雪靴/战斗靴（10%）、鹿皮裤/狼皮裤（10%）、战术手套（7%）、驼鹿皮斗篷（20%）、美洲狮皮围巾（6%）、冰爪（3%）和防弹背心（50%）。

### 机动性

服装提供的机动性惩罚会减少玩家可以冲刺的时间。当穿着多件导致机动性惩罚的服装时，惩罚是累积的（例如，一件减少机动性5%的外套和一双减少机动性4%的手套将总共减少9%的机动性）。

目前可以达到的最高机动性惩罚为-100%（技术上为-126%，但上限为-100%），使用以下组合：滑雪靴（-20%）、2条鹿皮裤（每条-5%）、2件考伊琴毛衣（每件-3%）、2件熊皮大衣（每件-20%）、一双兔皮手套（-5%）、冰爪（-5%）和防弹背心（-40%）。

### 耐久度

服装的耐久度会影响它的性能。随着服装的磨损，它对玩家的保暖和防风保护效果会降低。一件耐久度为50%的服装，如果保暖等级为+2.0度，则只会提供+1.0度的保暖效果。如果服装的耐久度降至0%，它将变为“损坏”，无法修复，但仍可以收获为原材料。

### 重量

服装的重量是指携带或放置在容器中时的重量。当穿着时，服装的重量只会对玩家的携带限制贡献75%。

## 分层

外层和内层服装的效果由数字顺序表示（最低数字最先受到影响，最高数字最后受到影响）。
